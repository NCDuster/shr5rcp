Fifth Edition Shadowrun Character Generators

The fast selection is a wizard :

  • core rule generator This generator is the priority system as defined by the core rule book.
  • Sum to Ten The sum to Ten generator from run faster.
  • life module The life module system from run faster.
  • point buy system This is a karma based generator from run faster.
  • freestyle A simple generator to create unconstrained character, like critter contact NPC, grunts, and so on.
